# README

Simple pointers example

## Creation

When building the code you have to initialise the module

```sh
go mod init github.com/chrisguest75/go_examples/04_runtime            
```

## Build and run

```sh
# build it
go build main.go

# run it
./main 
```

```sh
# build and run
go run main.go 
```

## Use documentation

```sh
go help modules 
go doc runtime
go doc os
```
